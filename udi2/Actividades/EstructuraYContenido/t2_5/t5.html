Clasificación de Elementos HTML

Elementos de Estilo

<i>: Este elemento se utiliza para representar texto en cursiva. Ejemplo en cursiva

<b>: Este elemento se utiliza para representar texto en negrita. Ejemplo en negrita

Etiquetas Semánticas

<strong>: Este elemento indica que el texto tiene una importancia fuerte. Ejemplo importante

<em>: Este elemento se utiliza para enfatizar el texto. Ejemplo enfatizado
