Atributos del elemento <area>

  • alt: Texto alternativo para el área. Importante para accesibilidad.
  • coords: Coordenadas del área en relación con la imagen en la que se utiliza. Su formato depende del valor de shape.
  • shape: Define la forma del área. Puede ser rect (rectángulo), circle (círculo), o poly (polígono).
  • href: URL de destino que se abrirá al hacer clic en el área.
  • target: Indica dónde abrir el enlace (_self, _blank, etc.).
  • rel: Relación entre el documento actual y el recurso vinculado. Usado comúnmente para indicar noopener o noreferrer para enlaces seguros.
